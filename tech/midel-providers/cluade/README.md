## Cluade의 가술을 실험하고 정리한다

### 지원하는 모델

- **Claude Opus 4.6**
  - 2026년 3월 기준으로 Anthropic 에서 개발한 가장 지능적인 모델
  - 깊은 추론, 미묘한 판단, 복잡한 멀티스텝 분석에 탁월
  - 대규모 리팩토링, 아키텍처 설계, 멀티에이전트 오케스트레이션, 연구 수준 분석
- **Claude Sonnet 4.6**
  - 강한 추론 능력, 좋은 코딩 실력, 안정적인 instruction following을 Opus 대비 훨씬 낮은 비용으로 제공
  - 대부분의 프로덕션 API 중심 사용 사례에서 가장 효율적
  - Anthropic 공식 권장: "모델 선택이 불확실하면 Sonnet으로 시작하라"
- **Claude Haiku 4.5**
  - 속도 최우선, Sonnet 대비 4~5배 빠르고 비용은 극히 저렴
  - 적합한 작업: 분류, 요약, 추출, 대량 API 호출, 챗봇 빠른 응답
  - 단순 분류/추출 작업에선 Sonnet 대비 정확도 차이가 2~5% 수준

### 지원하는 SDK

- 공식 홈페이지에서 2026년 3월 기준으로 Python, TypeScript, Java 지원
